# Community Sharing Platform

A React application built with Vite for carpooling and room sharing services.

## Features

- **Carpooling**: Post and search for rides.
- **Room Sharing**: Post and search for rooms for rent.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

To build the project for production:
```bash
npm run build
```

## Technologies

- React
- Vite
- ESLint
